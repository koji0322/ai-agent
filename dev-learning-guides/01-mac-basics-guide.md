# Mac 基礎操作ガイド

## 0. はじめに

このガイドは **Mac の基礎操作を体系的にまとめたリファレンス** です。

- **想定読者**: プログラミング初心者で、Mac の操作に不慣れな方
- **ゴール**: Mac の画面構成・キーボード・Finder・アプリ管理・システム設定を理解し、日常操作に困らなくなる
- **前提**: 特になし（どのレベルからでも参照可能）
- **関連ガイド**:
  - [VS Code 基本操作ガイド](00-vscode-guide.md) — エディタの使い方
  - [Mac おすすめアプリガイド](02-mac-apps-guide.md) — 開発に便利なアプリ
  - [ターミナル CLI ツールガイド](03-terminal-tools-guide.md) — コマンドラインの活用

> Mac は「メニューバーと Dock からすべてが始まる」OS。
> この考え方を押さえると、他の操作もスムーズに理解できる。

<details>
<summary>Windows ユーザー向けの補足</summary>

| 観点 | Windows | Mac |
|------|---------|-----|
| メニューの場所 | 各ウィンドウの上部 | **画面最上部に固定**（アクティブなアプリのメニュー） |
| アプリの終了 | ウィンドウの × で終了 | × はウィンドウを閉じるだけ（**アプリは終了しない**） |
| インストール | `.exe` を実行 | `.dmg` を開いてアプリを Applications にドラッグ |
| ファイルパス | `C:\Users\名前\Documents` | `/Users/名前/Documents` |
| 主な修飾キー | `Ctrl` | `Cmd`（⌘） |

</details>

## 1. 画面構成

Mac のデスクトップは大きく 3 つのエリアで構成される。

```
┌──────────────────────────────────────────────────────────────┐
│  リンゴ │ アプリメニュー        Wi-Fi  バッテリー  時計  Spotlight │ ← メニューバー
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                                                              │
│                     デスクトップ                                │
│                                                              │
│                                                              │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│    Finder  Safari  メール  ...              ゴミ箱             │ ← Dock
└──────────────────────────────────────────────────────────────┘
```

### 各エリアの役割

| エリア | 場所 | 役割 |
|-------|------|------|
| **メニューバー** | 画面最上部 | アクティブアプリのメニュー・システムアイコン |
| **デスクトップ** | 画面中央 | ファイルやフォルダを置ける作業領域 |
| **Dock** | 画面下部 | よく使うアプリのランチャー・実行中アプリの表示 |

### メニューバーの構成

| 位置 | 内容 | 説明 |
|------|------|------|
| 左端 |  メニュー | システム環境設定・スリープ・再起動・シャットダウン |
| 左側 | アプリメニュー | **アクティブなアプリ** のメニュー（ファイル・編集・表示…） |
| 右側 | ステータスメニュー | Wi-Fi・Bluetooth・バッテリー・時計・Spotlight |
| 右端 | 通知センター | 日付をクリックで開く。ウィジェット・通知を表示 |

> **重要**: メニューバーは **最前面のアプリに連動して変わる**。
> Finder が最前面なら Finder のメニュー、Safari が最前面なら Safari のメニューが表示される。

## 2. キーボード

### 修飾キー

| キー | 記号 | 備考 |
|------|------|------|
| `Cmd`（Command） | ⌘ | ほとんどのショートカットの主役 |
| `Option` | ⌥ | 特殊文字の入力・追加操作 |
| `Control` | ⌃ | ターミナル操作・右クリック（`Ctrl+クリック`） |
| `Shift` | ⇧ | 大文字入力・範囲選択 |
| `Fn`（Function） | 🌐 | ファンクションキーの切替・絵文字入力 |

<details>
<summary>Windows ユーザー向けの対応表</summary>

| Windows | Mac | 備考 |
|---------|-----|------|
| `Ctrl` | `Cmd`（Command） | ほとんどのショートカットで Ctrl → Cmd に読み替え |
| `Alt` | `Option` | 特殊文字の入力・追加操作 |
| `Windows` キー | `Cmd`（Command） | ランチャー的な用途は Spotlight が担う |

</details>

### 主要キーボードショートカット

#### 基本操作

| 操作 | ショートカット |
|------|-------------|
| コピー | `Cmd + C` |
| 貼り付け | `Cmd + V` |
| 切り取り | `Cmd + X` |
| 元に戻す | `Cmd + Z` |
| やり直し | `Cmd + Shift + Z` |
| 全選択 | `Cmd + A` |
| 保存 | `Cmd + S` |
| 検索 | `Cmd + F` |

#### Mac 固有の重要ショートカット

| 操作 | ショートカット | 説明 |
|------|-------------|------|
| Spotlight 検索 | `Cmd + Space` | アプリ起動・ファイル検索・計算など万能ランチャー |
| アプリ切替 | `Cmd + Tab` | 起動中のアプリ間を切り替え |
| 同一アプリ内の切替 | ``Cmd + ` `` | 同じアプリの別ウィンドウを切り替え |
| アプリ終了 | `Cmd + Q` | ウィンドウの × とは別。**アプリ自体を終了** |
| ウィンドウを閉じる | `Cmd + W` | ウィンドウだけ閉じる（アプリは残る） |
| 新しいウィンドウ | `Cmd + N` | 新しいウィンドウ / ドキュメント |
| 環境設定 | `Cmd + ,` | ほぼ全てのアプリで設定画面が開く |
| 絵文字 | `Fn + E` | 絵文字ピッカーを表示 |
| 画面ロック | `Cmd + Ctrl + Q` | 離席時にすぐロック |

### Delete キーの違い

| 操作 | Mac |
|------|-----|
| バックスペース（左の文字を消す） | `Delete` |
| デリート（右の文字を消す） | `Fn + Delete` |

> Mac の `Delete` キーは一般的な「Backspace」に相当する。右の文字を消したいときは `Fn + Delete` を使う。

## 3. トラックパッド

Mac のトラックパッドはジェスチャー操作が豊富。マウスがなくてもほとんどの操作が可能。

### 基本ジェスチャー

| ジェスチャー | 指の数 | 動作 |
|------------|--------|------|
| タップ | 1本指 | クリック |
| 2本指タップ | 2本指 | 右クリック（コンテキストメニュー） |
| 2本指スクロール | 2本指上下 | スクロール |
| ピンチイン / アウト | 2本指つまむ / 広げる | ズームイン / アウト |

### 応用ジェスチャー

| ジェスチャー | 指の数 | 動作 |
|------------|--------|------|
| 3本指上スワイプ | 3本指 | Mission Control（全ウィンドウ一覧） |
| 3本指左右スワイプ | 3本指 | デスクトップ（Spaces）の切り替え |
| 4本指ピンチ | 4本指つまむ | Launchpad（全アプリ一覧） |
| 3本指でドラッグ | 3本指 | ウィンドウやファイルのドラッグ |

> **3本指ドラッグの有効化**: デフォルトでは無効。
> `システム設定` → `アクセシビリティ` → `ポインタコントロール` → `トラックパッドオプション` → `ドラッグ方法` → `3本指のドラッグ` で有効にできる。

## 4. Finder（ファイル管理）

Finder は Mac に最初から入っているファイル管理アプリ。Dock の一番左にある **笑顔のアイコン** が Finder。

### Mac のフォルダ構造

```
/（ルート）
├── Users/           ← 全ユーザーのホームフォルダ
│   └── あなたの名前/  ← ホームフォルダ（🏠）
│       ├── Desktop/     ← デスクトップ
│       ├── Documents/   ← 書類
│       ├── Downloads/   ← ダウンロード
│       ├── Pictures/    ← ピクチャ
│       └── Movies/      ← ムービー
├── Applications/    ← アプリケーション
├── Library/         ← システムライブラリ（通常は触らない）
└── System/          ← macOS のシステムファイル（触らない）
```

> **ホームフォルダ** はターミナルでは `~`（チルダ）で表される。

### Finder の画面構成

```
┌──────────────────────────────────────────────┐
│ ◉ ◉ ◉  ← → │ 表示切替  │  検索バー        │  ← ツールバー
├──────┬───────────────────────────────────────┤
│      │                                       │
│サイド│  メインエリア                           │
│バー  │ （ファイル・フォルダの一覧）              │
│      │                                       │
├──────┴───────────────────────────────────────┤
│  パスバー（設定で表示）                        │
│  ステータスバー（設定で表示）                   │
└──────────────────────────────────────────────┘
```

### 表示スタイル

| 表示 | ショートカット | 特徴 | 向いている場面 |
|------|-------------|------|-------------|
| **アイコン** | `Cmd + 1` | サムネイルが大きく見える | 画像フォルダの閲覧 |
| **リスト** | `Cmd + 2` | ファイル名・日付・サイズが一覧で見える | 日常のファイル管理（おすすめ） |
| **カラム** | `Cmd + 3` | フォルダ階層を横に展開 | 深いフォルダ構造の探索 |
| **ギャラリー** | `Cmd + 4` | プレビューが大きく表示される | 写真の選別 |

### 基本操作

| やりたいこと | 操作 |
|------------|------|
| 開く | ダブルクリック or `Cmd + O` |
| 新規フォルダ作成 | `Cmd + Shift + N` |
| コピー | `Cmd + C` → `Cmd + V` |
| **移動** | `Cmd + C` → **`Cmd + Option + V`** |
| 名前変更 | ファイル選択 → `Enter` |
| 削除（ゴミ箱へ） | `Cmd + Delete` |
| 情報を見る | `Cmd + I` |
| クイックルック | `Space` キー |

> **重要**: Mac では `Enter` キーは「開く」ではなく「名前変更」。
> ファイルを開くのはダブルクリックまたは `Cmd + O`。

### ナビゲーション

| やりたいこと | 操作 |
|------------|------|
| 一つ上のフォルダへ | `Cmd + ↑` |
| フォルダを開く | `Cmd + ↓` またはダブルクリック |
| 戻る | `Cmd + [` |
| 進む | `Cmd + ]` |
| ホームフォルダへ | `Cmd + Shift + H` |
| デスクトップへ | `Cmd + Shift + D` |
| ダウンロードへ | `Cmd + Option + L` |
| アプリケーションへ | `Cmd + Shift + A` |
| パスを指定して移動 | `Cmd + Shift + G`（パスを直接入力） |

> **`Cmd + Shift + G` は非常に便利**。ターミナルで見たパスをそのまま貼り付けて移動できる。

### サイドバーの活用

サイドバーには「よく使う項目」「iCloud」「場所」「タグ」の各セクションがある。

**自分のフォルダを追加する方法**: 追加したいフォルダをサイドバーの「よく使う項目」にドラッグする。

> プロジェクトフォルダをサイドバーに追加しておくと、一発でアクセスできて便利。

### Finder のおすすめ設定

| 設定 | 方法 | 効果 |
|------|------|------|
| パスバー表示 | `Cmd + Option + P` | 現在の場所をパスで表示 |
| ステータスバー表示 | `Cmd + /` | 項目数・空き容量を表示 |
| 拡張子を常に表示 | Finder 設定 → 詳細 → すべてのファイル名拡張子を表示 | `.py` `.js` が常に見える |
| 隠しファイル表示 | `Cmd + Shift + .` | `.gitignore` 等を確認するときに必要 |
| タイトルにパス表示 | ターミナルで `defaults write com.apple.finder _FXShowPosixPathInTitle -bool true && killall Finder` | 完全なパスが表示される |

### Finder とターミナルの連携

| やりたいこと | 方法 |
|------------|------|
| ターミナルから Finder を開く | `open .`（現在のフォルダを開く） |
| Finder のフォルダをターミナルへ | フォルダをターミナルにドラッグ（パスが自動入力） |
| VS Code から Finder を開く | ファイルを右クリック → Reveal in Finder |

> `open .` は非常によく使うコマンド。ターミナルで作業中に「このフォルダを Finder で見たい」というときに便利。

## 5. Spotlight 検索

Spotlight は Mac 全体の万能ランチャー。`Cmd + Space` で即座に呼び出せる。

### Spotlight でできること

| 機能 | 入力例 | 説明 |
|------|--------|------|
| アプリ起動 | `ter` → Terminal | 名前の一部を打つだけで候補が出る |
| ファイル検索 | `レポート.xlsx` | ファイル名で検索 |
| 計算 | `1280 * 1.1` | 計算結果をその場で表示 |
| 単位変換 | `100 usd to jpy` | 通貨・温度・長さなどの変換 |
| 辞書 | `define ephemeral` | 単語の意味を表示 |
| Web 検索 | 検索ワードを入力 → Enter | Safari で検索結果を表示 |
| システム設定 | `bluetooth` | 該当する設定画面を直接開く |

> **最も重要な使い方**: アプリの起動。
> `Cmd + Space` → アプリ名の数文字 → `Enter` が Mac で最速のアプリ起動方法。

## 6. アプリの基本

### アプリの起動方法

| 方法 | 操作 | おすすめ度 |
|------|------|----------|
| **Spotlight** | `Cmd + Space` → アプリ名を入力 | ★★★（最速） |
| **Dock** | Dock のアイコンをクリック | ★★☆ |
| **Launchpad** | Dock の Launchpad アイコン or 4本指ピンチ | ★☆☆ |
| **Finder** | `/Applications` フォルダを開く | ★☆☆ |

### アプリの終了（重要！）

| 操作 | 方法 | 結果 |
|------|------|------|
| ウィンドウを閉じる | 赤い × ボタン / `Cmd + W` | **ウィンドウだけ閉じる。アプリは動いたまま** |
| **アプリを終了** | `Cmd + Q` | **アプリ自体を終了** |
| Dock から終了 | Dock のアイコンを右クリック → 「終了」 | アプリ自体を終了 |

> Mac は × ボタンを押してもアプリは **バックグラウンドで動き続ける**。
> 完全に終了するには `Cmd + Q` を使う。
> Dock でアイコンの下に **小さな点** があるアプリが実行中。

### アプリの強制終了

アプリがフリーズした場合: `Cmd + Option + Esc` → アプリを選択 → 「強制終了」

### アプリのインストール

Mac でアプリをインストールする方法は主に 3 つ。

| 方法 | 安全性 | 手軽さ | 説明 |
|------|--------|--------|------|
| **App Store** | ★★★ | ★★★ | Apple 公式ストア。審査済みで安全 |
| **dmg / pkg** | ★★☆ | ★★☆ | 開発元サイトからダウンロード |
| **Homebrew** | ★★☆ | ★★★（慣れれば） | ターミナルからインストール |

#### Homebrew（brew）

ターミナルからアプリを管理するパッケージマネージャー。開発者向けだが非常に便利。

```bash
# Homebrew のインストール（初回のみ）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# アプリのインストール例
brew install git                        # CLI ツール
brew install --cask visual-studio-code  # GUI アプリ

# アップデート
brew update && brew upgrade             # 全体を最新に更新
```

| コマンド | 説明 |
|---------|------|
| `brew install パッケージ名` | CLI ツールをインストール |
| `brew install --cask アプリ名` | GUI アプリをインストール |
| `brew list` | インストール済み一覧 |
| `brew uninstall パッケージ名` | アンインストール |

### アプリのアンインストール

| 方法 | 操作 |
|------|------|
| Launchpad | アイコンを長押し → × ボタン（App Store アプリのみ） |
| Finder | `/Applications` からゴミ箱にドラッグ |
| Homebrew | `brew uninstall --cask アプリ名` |

## 7. Dock

Dock は画面下部に表示されるアプリのランチャー兼タスクバー。

```
┌─────────────────────────────────────────────────────────┐
│  Finder  Safari  メール  ...  │  最近使ったアプリ  │  ゴミ箱  │
│  ← アプリエリア →             │  ← 最近使用 →     │         │
└─────────────────────────────────────────────────────────┘
```

### アプリの追加 / 削除

| 操作 | 方法 |
|------|------|
| 追加 | アプリを Dock にドラッグ / 実行中のアプリを右クリック → 「Dock に残す」 |
| 削除 | Dock のアイコンをドラッグして外に出す |
| 並び替え | アイコンをドラッグして好きな位置へ |

### おすすめ設定

`システム設定 → デスクトップと Dock` で変更。

| 設定 | おすすめ | 理由 |
|------|---------|------|
| サイズ | 小さめ | 画面を広く使える |
| 拡大 | オン | マウスを乗せるとアイコンが拡大して見やすい |
| 自動的に表示/非表示 | オン | 普段は隠れて、マウスを下に持っていくと出てくる |

## 8. Mission Control とウィンドウ管理

### Mission Control

全ウィンドウを俯瞰する機能。

| 起動方法 | 操作 |
|---------|------|
| トラックパッド | 3本指上スワイプ |
| キーボード | `F3`（または `Ctrl + ↑`） |

### 仮想デスクトップ（Spaces）

複数のデスクトップを作成し、作業を分けることができる。

| 操作 | 方法 |
|------|------|
| デスクトップの切替 | 3本指左右スワイプ / `Ctrl + ←` / `Ctrl + →` |
| 新しいデスクトップを作成 | Mission Control → 右上の「＋」ボタン |
| ウィンドウを別のデスクトップへ | Mission Control でウィンドウを上部のデスクトップにドラッグ |

> **使い方の例**:
> - デスクトップ 1: メイン作業（VS Code + ターミナル）
> - デスクトップ 2: ブラウザ（調べもの）
> - デスクトップ 3: メール・チャット

### Split View（画面分割）

ウィンドウの緑ボタン（◉）を長押し → 「画面左側にタイル表示」/「右側」で、2 つのアプリを左右に並べて表示できる。

## 9. スクリーンショット

Mac にはスクリーンショット機能が標準搭載されている（追加アプリ不要）。

| 操作 | ショートカット | 説明 |
|------|-------------|------|
| **画面全体** | `Cmd + Shift + 3` | 全画面をキャプチャ |
| **範囲選択** | `Cmd + Shift + 4` → ドラッグ | 選択範囲をキャプチャ |
| **ウィンドウ** | `Cmd + Shift + 4` → `Space` → クリック | 特定ウィンドウをキャプチャ |
| **ツールバー** | `Cmd + Shift + 5` | 各種オプション・画面録画にも対応 |

> クリップボードにコピーしたい場合は `Ctrl` を追加（例: `Cmd + Ctrl + Shift + 4`）

## 10. システム設定

macOS の設定は「システム設定」で変更する。

### 開き方

 メニュー → 「システム設定」、または `Cmd + Space` → 「システム設定」

### 最初に変更すべき設定項目

| 設定場所 | 項目 | おすすめ |
|---------|------|---------|
| トラックパッド → ポイントとクリック | タップでクリック | **オン** |
| トラックパッド → ポイントとクリック | 軌跡の速さ | **速め** |
| アクセシビリティ → ポインタコントロール | 3本指ドラッグ | **オン** |
| キーボード | キーのリピート速度 | **速い** |
| キーボード | リピート入力認識までの時間 | **短い** |
| Dock | 自動的に表示/非表示 | **オン** |
| 一般 → AirDrop と Handoff | AirDrop | 連絡先のみ |

### ホットコーナーのおすすめ設定

`システム設定 → デスクトップと Dock → ホットコーナー`

| コーナー | おすすめアクション |
|---------|-----------------|
| 左上 | Mission Control |
| 右上 | デスクトップ |
| 左下 | （なし） |
| 右下 | 画面ロック |

## 11. セキュリティ

### FileVault（ディスク暗号化）

Mac のストレージ全体を暗号化する機能。万が一盗まれてもデータを読み取れなくする。

`システム設定 → プライバシーとセキュリティ → FileVault` で確認・有効化。

> **推奨**: Mac を持ち歩く場合は必ずオンにすべき。

### Gatekeeper（アプリの実行制限）

`システム設定 → プライバシーとセキュリティ → セキュリティ`

| 設定 | 意味 |
|------|------|
| App Store | App Store のアプリのみ許可（最も厳しい） |
| App Store と確認済みの開発元 | **おすすめ**。署名付きアプリも許可 |

> 「開発元を検証できません」と表示されたら、`システム設定` → `プライバシーとセキュリティ` の下部にある「許可」ボタンから開ける。

### パスワード管理

| 項目 | 説明 |
|------|------|
| ログインパスワード | Mac 起動時・スリープ復帰時に入力 |
| Apple ID | App Store・iCloud・各種サービスで使用 |
| キーチェーン | Mac 標準のパスワード管理機能 |
| Touch ID | 指紋認証でパスワード入力を省略 |

## 12. バックアップ（Time Machine）

Mac 標準のバックアップ機能。外付けドライブに自動でバックアップを取り続ける。

| 項目 | 内容 |
|------|------|
| 何をバックアップ | Mac 全体（アプリ・設定・ファイルすべて） |
| 頻度 | 1 時間ごとに自動 |
| 必要なもの | **外付けドライブ**（USB / Thunderbolt） |

### 設定手順

1. 外付けドライブを Mac に接続
2. 「Time Machine バックアップとして使用しますか？」→ 「使用」
3. 初回バックアップが自動で開始（時間がかかる）
4. 以降は接続するたびに自動バックアップ

### ファイルの復元

メニューバーの Time Machine アイコン → 「Time Machine に入る」 → タイムラインを遡って復元。

## 13. トラブルシューティング

### アプリがフリーズした

`Cmd + Option + Esc` → フリーズしたアプリを選択 → 「強制終了」

### Mac 全体がフリーズした

1. 数分待つ（重い処理中の可能性がある）
2. 電源ボタンを長押し（10秒）で **強制シャットダウン**（最終手段）
3. 数秒待ってから電源ボタンで起動

### ディスク容量が足りない

|  →「この Mac について」→「ストレージ」で容量を確認。ゴミ箱を空にする、不要な `.dmg` ファイルを削除するのが最初の対処。

### セーフモード（Apple Silicon Mac）

1. Mac をシャットダウン
2. 電源ボタンを長押し（「起動オプションを読み込み中」が表示されるまで）
3. 起動ディスクを選択
4. `Shift` キーを押しながら「セーフモードで続ける」をクリック

> セーフモードで問題が発生しない場合、サードパーティのソフトが原因の可能性が高い。

### よくあるトラブル

| 症状 | 対処法 |
|------|--------|
| Wi-Fi がつながらない | Wi-Fi をオフ → オン / ルーターを再起動 |
| Bluetooth が不安定 | デバイスを削除して再ペアリング |
| 動作が重い | アクティビティモニタで CPU / メモリ使用量を確認 |
| 「このアプリは壊れています」 | ターミナルで `xattr -cr /Applications/アプリ名.app` |
| `.DS_Store` が邪魔 | `.gitignore` に追加: `echo ".DS_Store" >> .gitignore` |
