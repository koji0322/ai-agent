# Mac 基礎操作ガイド

## 0. はじめに

このガイドは **Mac の基礎操作を体系的にまとめた自分用リファレンス** です。

- **想定読者**: Windows は使えるが、Mac に移行したばかりで操作に不慣れ
- **ゴール**: Mac の画面構成・キーボード・アプリ管理・システム設定を理解し、日常操作に困らなくなる
- **前提**: 特になし（どのレベルからでも参照可能）
- **関連ガイド**: [08 - Mac Finder ガイド](08-mac-finder-guide.md)（ファイル管理の詳細）

> Windows は「スタートメニューからすべてが始まる」OS。
> Mac は「メニューバーとDockからすべてが始まる」OS。
> 根本的な考え方の違いを押さえると、他の操作もスムーズに理解できる。

### Windows と Mac の根本的な違い

| 観点 | Windows | Mac |
|------|---------|-----|
| メニューの場所 | 各ウィンドウの上部 | **画面最上部に固定**（アクティブなアプリのメニュー） |
| アプリの終了 | ウィンドウの × で終了 | × はウィンドウを閉じるだけ（**アプリは終了しない**） |
| インストール | `.exe` を実行 | `.dmg` を開いてアプリを Applications にドラッグ |
| ファイルパス | `C:\Users\名前\Documents` | `/Users/名前/Documents` |
| 主な修飾キー | `Ctrl` | `Cmd`（⌘） |

## 1. 画面構成

Mac のデスクトップは大きく 3 つのエリアで構成される。

```
┌──────────────────────────────────────────────────────────────┐
│  リンゴ │ アプリメニュー        Wi-Fi  バッテリー  時計  Spotlight │ ← メニューバー
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                                                              │
│                     デスクトップ                                │
│                                                              │
│                                                              │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│    Finder  Safari  メール  ...              ゴミ箱             │ ← Dock
└──────────────────────────────────────────────────────────────┘
```

### 各エリアの役割

| エリア | 場所 | 役割 | Windows で例えると |
|-------|------|------|------------------|
| **メニューバー** | 画面最上部 | アクティブアプリのメニュー・システムアイコン | タスクバー右側 + 各ウィンドウのメニュー |
| **デスクトップ** | 画面中央 | ファイルやフォルダを置ける作業領域 | デスクトップ（同じ） |
| **Dock** | 画面下部 | よく使うアプリのランチャー・実行中アプリの表示 | タスクバー |

### メニューバーの構成

| 位置 | 内容 | 説明 |
|------|------|------|
| 左端 |  メニュー | システム環境設定・スリープ・再起動・シャットダウン |
| 左側 | アプリメニュー | **アクティブなアプリ** のメニュー（ファイル・編集・表示…） |
| 右側 | ステータスメニュー | Wi-Fi・Bluetooth・バッテリー・時計・Spotlight |
| 右端 | 通知センター | 日付をクリックで開く。ウィジェット・通知を表示 |

> **重要**: メニューバーは **最前面のアプリに連動して変わる**。
> Finder が最前面なら Finder のメニュー、Safari が最前面なら Safari のメニューが表示される。

## 2. キーボード

### 修飾キーの対応（Windows → Mac）

| Windows | Mac | 記号 | 備考 |
|---------|-----|------|------|
| `Ctrl` | `Cmd`（Command） | ⌘ | ほとんどのショートカットの主役 |
| `Alt` | `Option` | ⌥ | 特殊文字の入力・追加操作 |
| `Windows` キー | `Cmd`（Command） | ⌘ | ランチャー的な用途は Spotlight が担う |
| `Ctrl`（残る） | `Control` | ⌃ | ターミナル操作・右クリック（`Ctrl+クリック`） |
| `Shift` | `Shift` | ⇧ | 同じ |
| — | `Fn`（Function） | 🌐 | ファンクションキーの切替・絵文字入力 |

> **最重要ルール**: Windows の `Ctrl` → Mac の `Cmd` に読み替えればほぼ OK。
> 例: `Ctrl+C`（コピー）→ `Cmd+C`、`Ctrl+V`（貼り付け）→ `Cmd+V`

### 記号の読み方（メニューでよく見る）

```
⌘  Command（Cmd）
⌥  Option（Alt）
⌃  Control（Ctrl）
⇧  Shift
⏎  Return（Enter）
⌫  Delete（バックスペース）
⎋  Escape
```

### 主要キーボードショートカット

#### 基本操作（Windows とほぼ同じ、Ctrl → Cmd に変更）

| 操作 | Mac | Windows（参考） |
|------|-----|----------------|
| コピー | `Cmd + C` | `Ctrl + C` |
| 貼り付け | `Cmd + V` | `Ctrl + V` |
| 切り取り | `Cmd + X` | `Ctrl + X` |
| 元に戻す | `Cmd + Z` | `Ctrl + Z` |
| やり直し | `Cmd + Shift + Z` | `Ctrl + Y` |
| 全選択 | `Cmd + A` | `Ctrl + A` |
| 保存 | `Cmd + S` | `Ctrl + S` |
| 印刷 | `Cmd + P` | `Ctrl + P` |
| 検索 | `Cmd + F` | `Ctrl + F` |

#### Mac 固有の重要ショートカット

| 操作 | ショートカット | 説明 |
|------|-------------|------|
| Spotlight 検索 | `Cmd + Space` | アプリ起動・ファイル検索・計算など万能ランチャー |
| アプリ切替 | `Cmd + Tab` | Windows の `Alt + Tab` に相当 |
| ウィンドウ切替 | ``Cmd + ` ``（バッククォート） | 同じアプリ内のウィンドウ切替 |
| アプリ終了 | `Cmd + Q` | ウィンドウの × とは別。**アプリ自体を終了** |
| ウィンドウを閉じる | `Cmd + W` | ウィンドウだけ閉じる（アプリは残る） |
| 新しいウィンドウ | `Cmd + N` | 新しいウィンドウ / ドキュメント |
| 環境設定 | `Cmd + ,`（カンマ） | ほぼ全てのアプリで設定画面が開く |
| 絵文字・特殊文字 | `Fn + E` または `Ctrl + Cmd + Space` | 絵文字ピッカーを表示 |
| 画面ロック | `Cmd + Ctrl + Q` | 離席時にすぐロック |

### Delete キーの違い

| 操作 | Mac | Windows |
|------|-----|---------|
| バックスペース（左の文字を消す） | `Delete` | `Backspace` |
| デリート（右の文字を消す） | `Fn + Delete` | `Delete` |

> Mac の `Delete` キーは Windows の `Backspace` に相当する。
> 右の文字を消したいときは `Fn + Delete` を使う。

## 3. トラックパッド

Mac のトラックパッドはジェスチャー操作が豊富。マウスがなくてもほとんどの操作が可能。

### 基本ジェスチャー

| ジェスチャー | 指の数 | 動作 | Windows で例えると |
|------------|--------|------|------------------|
| タップ | 1本指 | クリック | 左クリック |
| 2本指タップ | 2本指 | 右クリック（コンテキストメニュー） | 右クリック |
| 2本指スクロール | 2本指上下 | スクロール | マウスホイール |
| ピンチイン / アウト | 2本指つまむ / 広げる | ズームイン / アウト | `Ctrl + マウスホイール` |

### 応用ジェスチャー

| ジェスチャー | 指の数 | 動作 |
|------------|--------|------|
| 3本指上スワイプ | 3本指 | Mission Control（全ウィンドウ一覧） |
| 3本指左右スワイプ | 3本指 | デスクトップ（Spaces）の切り替え |
| 4本指ピンチ | 4本指つまむ | Launchpad（全アプリ一覧） |
| 3本指でドラッグ | 3本指 | ウィンドウやファイルのドラッグ |

> **3本指ドラッグの有効化**: デフォルトでは無効。
> `システム設定` → `アクセシビリティ` → `ポインタコントロール` → `トラックパッドオプション` → `ドラッグ方法` → `3本指のドラッグ` で有効にできる。

### スクロールの方向（ナチュラルスクロール）

Mac のデフォルトは **ナチュラルスクロール**（2本指を上に動かすとコンテンツが上に動く = スマホと同じ方向）。
Windows のマウスホイールとは逆方向なので、違和感がある場合は設定で変更可能。

```
システム設定 → トラックパッド → スクロールとズーム → ナチュラルなスクロール
```

## 4. アプリの基本

### アプリの起動方法

| 方法 | 操作 | おすすめ度 |
|------|------|----------|
| **Spotlight** | `Cmd + Space` → アプリ名を入力 | ★★★（最速） |
| **Dock** | Dock のアイコンをクリック | ★★☆ |
| **Launchpad** | Dock の Launchpad アイコン or 4本指ピンチ | ★☆☆ |
| **Finder** | `/Applications` フォルダを開く | ★☆☆ |

> **おすすめ**: Spotlight（`Cmd + Space`）で名前の最初の数文字を打つのが最速。
> 例: 「ter」と打てば Terminal が候補に出る。

### アプリの切り替え

| 操作 | ショートカット | 説明 |
|------|-------------|------|
| アプリ間の切替 | `Cmd + Tab` | Tab を押すたびに次のアプリ。押したまま左右で選択 |
| 同一アプリ内の切替 | ``Cmd + ` `` | 同じアプリの別ウィンドウを切り替え |
| Mission Control | `F3` or 3本指上スワイプ | 全ウィンドウを俯瞰して選択 |

### アプリの終了（重要！）

| 操作 | 方法 | 結果 |
|------|------|------|
| ウィンドウを閉じる | 赤い × ボタン / `Cmd + W` | **ウィンドウだけ閉じる。アプリは動いたまま** |
| **アプリを終了** | `Cmd + Q` | **アプリ自体を終了** |
| メニューから終了 | メニューバー → アプリ名 → 「終了」 | アプリ自体を終了 |
| Dock から終了 | Dock のアイコンを右クリック →「終了」 | アプリ自体を終了 |

> **Windows との最大の違い**: Windows は × ボタンでアプリが終了するが、
> Mac は × ボタンを押してもアプリは **バックグラウンドで動き続ける**。
> 完全に終了するには `Cmd + Q` を使う。
> Dock でアイコンの下に **小さな点** があるアプリが実行中。

### アプリの強制終了

アプリがフリーズした場合の対処法。

| 方法 | 操作 |
|------|------|
| ショートカット | `Cmd + Option + Esc` → アプリを選択 →「強制終了」 |
|  メニュー | 左上  →「強制終了」 |
| Dock | アイコンを `Option` + 右クリック →「強制終了」 |

## 5. Dock

Dock は画面下部に表示されるアプリのランチャー兼タスクバー。

### Dock の構成

```
┌─────────────────────────────────────────────────────────┐
│  Finder  Safari  メール  ...  │  最近使ったアプリ  │  ゴミ箱  │
│  ← アプリエリア →             │  ← 最近使用 →     │         │
└─────────────────────────────────────────────────────────┘
```

| エリア | 内容 |
|-------|------|
| アプリエリア（左側） | お気に入り登録したアプリ・実行中のアプリ |
| 最近使ったアプリ（中央） | 最近起動したアプリ（3つ程度） |
| ファイル・ゴミ箱（右側） | ダウンロードフォルダ・ゴミ箱 |

### Dock のアイコン表示

| 状態 | 表示 |
|------|------|
| 実行中 | アイコンの下に **小さな点（●）** |
| 未実行 | 点なし |

### アプリの追加 / 削除

| 操作 | 方法 |
|------|------|
| 追加 | アプリを Dock にドラッグ / 実行中のアプリを右クリック →「オプション」→「Dock に残す」 |
| 削除 | Dock のアイコンをドラッグして外に出す（「削除」のラベルが出る） |
| 並び替え | アイコンをドラッグして好きな位置へ |

### Dock のおすすめ設定

```
システム設定 → デスクトップと Dock
```

| 設定 | おすすめ | 説明 |
|------|---------|------|
| サイズ | 小さめ | 画面を広く使える |
| 拡大 | オン | マウスを乗せるとアイコンが拡大して見やすい |
| 自動的に表示/非表示 | オン | 普段は隠れて、マウスを下に持っていくと出てくる |
| 最近使ったアプリを表示 | お好みで | Dock が増えるのが嫌ならオフ |

## 6. Mission Control

Mission Control は Mac のウィンドウ管理の中核機能。

### 起動方法

| 方法 | 操作 |
|------|------|
| トラックパッド | 3本指上スワイプ |
| キーボード | `F3`（または `Ctrl + ↑`） |
| マウス | ホットコーナーに割り当て |

### 仮想デスクトップ（Spaces）

Mac では複数のデスクトップ（Spaces）を作成し、作業を分けることができる。

| 操作 | 方法 |
|------|------|
| デスクトップの切替 | 3本指左右スワイプ / `Ctrl + ←` / `Ctrl + →` |
| 新しいデスクトップを作成 | Mission Control → 右上の「＋」ボタン |
| デスクトップの削除 | Mission Control → デスクトップにカーソルを乗せて × |
| ウィンドウを別のデスクトップへ | Mission Control でウィンドウを上部のデスクトップにドラッグ |

> **使い方の例**:
> - デスクトップ 1: メイン作業（VS Code + ターミナル）
> - デスクトップ 2: ブラウザ（調べもの）
> - デスクトップ 3: メール・チャット

### Split View（画面分割）

1 つのデスクトップで 2 つのアプリを左右に並べて表示する機能。

| 操作 | 方法 |
|------|------|
| Split View に入る | ウィンドウの緑ボタン（◉）を長押し →「画面左側にタイル表示」/「右側」 |
| 分割比率の変更 | 中央の境界線をドラッグ |
| Split View を解除 | 緑ボタンをクリック / `Esc` |

### ウィンドウの緑ボタン操作

ウィンドウ左上の緑ボタン（◉）は複数の機能を持つ。

| 操作 | 結果 |
|------|------|
| クリック | フルスクリーン |
| `Option` + クリック | 画面いっぱいに拡大（フルスクリーンではない） |
| 長押し | タイル表示のオプション（Split View） |

## 7. Spotlight 検索

Spotlight は Mac 全体の万能ランチャー。`Cmd + Space` で即座に呼び出せる。

### 起動方法

| 方法 | 操作 |
|------|------|
| キーボード | `Cmd + Space` |
| メニューバー | 右上の 🔍 アイコン |

### Spotlight でできること

| 機能 | 入力例 | 説明 |
|------|--------|------|
| アプリ起動 | `ter` → Terminal | 名前の一部を打つだけで候補が出る |
| ファイル検索 | `レポート.xlsx` | ファイル名で検索 |
| 計算 | `1280 * 1.1` | 計算結果をその場で表示 |
| 単位変換 | `100 usd to jpy` | 通貨・温度・長さなどの変換 |
| 辞書 | `define ephemeral` | 単語の意味を表示 |
| Web 検索 | 検索ワードを入力 → Enter | Safari で検索結果を表示 |
| システム設定 | `bluetooth` | 該当する設定画面を直接開く |

> **最も重要な使い方**: アプリの起動。スタートメニューやデスクトップショートカットの代わりに、
> `Cmd + Space` → アプリ名の数文字 → `Enter` が Mac で最速のアプリ起動方法。

> **Finder 内の検索との違い**: Finder の検索（`Cmd + F`）はファイルの詳細検索向き。
> Spotlight はアプリ起動・簡易検索・計算など「何でも屋」。
> 詳しくは [08 - Mac Finder ガイド](08-mac-finder-guide.md) を参照。

## 8. スクリーンショット

Mac にはスクリーンショット機能が標準搭載されている（追加アプリ不要）。

### スクリーンショットの撮影

| 操作 | ショートカット | 保存先 |
|------|-------------|--------|
| **画面全体** | `Cmd + Shift + 3` | デスクトップ |
| **範囲選択** | `Cmd + Shift + 4` → ドラッグで範囲指定 | デスクトップ |
| **ウィンドウ** | `Cmd + Shift + 4` → `Space` → ウィンドウをクリック | デスクトップ |
| **ツールバー表示** | `Cmd + Shift + 5` | 設定による |

### スクリーンショットツールバー（`Cmd + Shift + 5`）

`Cmd + Shift + 5` を押すと画面下部にツールバーが表示され、以下の操作が可能。

| ボタン | 機能 |
|-------|------|
| 画面全体をキャプチャ | 全画面のスクリーンショット |
| ウィンドウをキャプチャ | 特定のウィンドウだけ |
| 範囲をキャプチャ | ドラッグして選択した範囲 |
| 画面全体を収録 | 画面全体の動画を録画 |
| 範囲を収録 | 選択した範囲の動画を録画 |
| オプション | 保存先・タイマー・マウスカーソルの表示設定 |

### 便利なテクニック

| テクニック | 操作 |
|-----------|------|
| クリップボードにコピー（ファイル保存しない） | 各ショートカットに `Ctrl` を追加（例: `Cmd + Ctrl + Shift + 4`） |
| 撮影後すぐ編集 | 右下のサムネイルをクリック → マークアップで注釈追加 |
| 保存先の変更 | `Cmd + Shift + 5` → オプション → 保存先を変更 |

## 9. アプリのインストール

Mac でアプリをインストールする方法は主に 3 つ。

### 方法の比較

| 方法 | 安全性 | 手軽さ | 説明 |
|------|--------|--------|------|
| **App Store** | ★★★ | ★★★ | Apple 公式ストア。審査済みで安全 |
| **dmg / pkg** | ★★☆ | ★★☆ | 開発元サイトからダウンロード |
| **Homebrew** | ★★☆ | ★★★（慣れれば） | ターミナルからインストール |

### App Store

1. Dock の「App Store」アイコンをクリック（または Spotlight で `app store`）
2. アプリを検索
3. 「入手」→「インストール」をクリック

> Windows の Microsoft Store と同じ仕組み。

### dmg ファイル（ディスクイメージ）

多くの Mac アプリは `.dmg` 形式で配布される。

```
1. ブラウザで .dmg をダウンロード
2. ダウンロードした .dmg をダブルクリック → 仮想ディスクがマウントされる
3. 表示されたウィンドウでアプリアイコンを Applications フォルダにドラッグ
4. マウントされた仮想ディスクを取り出す（Finder サイドバーの ⏏ アイコン）
```

> **Windows との違い**: Windows の `.exe` はインストーラーが自動で進むが、
> Mac の `.dmg` は「アプリを Applications に入れる」という手動操作が多い。
> `.pkg` ファイルの場合はウィザード形式のインストーラーが起動する（Windows に近い）。

### Homebrew（brew）

ターミナルからアプリを管理するパッケージマネージャー。開発者向けだが非常に便利。

```bash
# Homebrew のインストール（初回のみ）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# アプリのインストール例
brew install git           # CLI ツール
brew install --cask visual-studio-code  # GUI アプリ

# アップデート
brew update                # Homebrew 自体を更新
brew upgrade               # インストール済みアプリを一括更新

# 検索
brew search chrome         # アプリを検索
```

| コマンド | 説明 |
|---------|------|
| `brew install パッケージ名` | CLI ツールをインストール |
| `brew install --cask アプリ名` | GUI アプリをインストール |
| `brew list` | インストール済み一覧 |
| `brew uninstall パッケージ名` | アンインストール |
| `brew update && brew upgrade` | 全体を最新に更新 |

### アプリのアンインストール

| 方法 | 操作 |
|------|------|
| Launchpad | アイコンを長押し → × ボタン（App Store アプリのみ） |
| Finder | `/Applications` からゴミ箱にドラッグ |
| Homebrew | `brew uninstall --cask アプリ名` |

> **Windows との違い**: Windows の「プログラムの追加と削除」のような統一的な仕組みはない。
> Finder から Applications のアプリをゴミ箱に入れるのが基本的な方法。

## 10. システム設定

macOS の設定は「システム設定」（macOS Ventura 以降）で変更する。

### 開き方

| 方法 | 操作 |
|------|------|
|  メニュー | 左上  →「システム設定」 |
| Spotlight | `Cmd + Space` → `システム設定` |
| Dock | 歯車アイコン（あれば） |

> Windows の「設定」/「コントロールパネル」に相当する。

### 最初に変更すべき設定項目

| 設定場所 | 項目 | おすすめ設定 | 理由 |
|---------|------|------------|------|
| トラックパッド → ポイントとクリック | タップでクリック | **オン** | 押し込まなくてもタップでクリックできる |
| トラックパッド → ポイントとクリック | 軌跡の速さ | **速め** | カーソル移動が速くなる |
| アクセシビリティ → ポインタコントロール | 3本指ドラッグ | **オン** | ウィンドウやファイルのドラッグが楽になる |
| キーボード | キーのリピート速度 | **速い** | 文字の連続入力が速くなる |
| キーボード | リピート入力認識までの時間 | **短い** | キーを押してからリピートが始まるまでが速い |
| Dock | 自動的に表示/非表示 | **オン** | 画面を広く使える |
| Dock | 拡大 | **オン** | アイコンにカーソルを乗せると拡大 |
| デスクトップと Dock | ホットコーナー | お好みで | 画面の四隅にアクションを割り当て |
| 一般 → AirDrop と Handoff | AirDrop | 連絡先のみ | セキュリティのため |
| バッテリー | ディスプレイのオフタイマー | お好みで | バッテリー節約 |

### ホットコーナーのおすすめ設定

画面の四隅にマウスを持っていくと特定のアクションを実行する機能。

```
システム設定 → デスクトップと Dock → ホットコーナー
```

| コーナー | おすすめアクション |
|---------|-----------------|
| 左上 | Mission Control |
| 右上 | デスクトップ |
| 左下 | （なし） |
| 右下 | 画面ロック |

## 11. セキュリティ

### FileVault（ディスク暗号化）

Mac のストレージ全体を暗号化する機能。万が一盗まれてもデータを読み取れなくする。

```
システム設定 → プライバシーとセキュリティ → FileVault
```

| 項目 | 説明 |
|------|------|
| 状態確認 | 「FileVault がオンです」と表示されていれば有効 |
| 有効化 | 「オンにする」をクリック → 復旧キーを安全な場所に保管 |
| 復旧キー | **絶対に紛失しないこと**。パスワードを忘れた際の唯一の救済手段 |

> **推奨**: Mac を持ち歩く場合は必ずオンにすべき。
> Windows の BitLocker に相当する。

### Gatekeeper（アプリの実行制限）

信頼できないアプリの実行をブロックする仕組み。

```
システム設定 → プライバシーとセキュリティ → セキュリティ
```

| 設定 | 意味 |
|------|------|
| App Store | App Store のアプリのみ許可（最も厳しい） |
| App Store と確認済みの開発元 | **おすすめ**。署名付きアプリも許可 |

> **「開発元を検証できません」と表示されたら**:
> そのアプリを信頼できる場合、`システム設定` → `プライバシーとセキュリティ` → 下部に「〇〇のアプリを許可」ボタンが表示される。

### パスワード管理

| 項目 | 説明 |
|------|------|
| ログインパスワード | Mac 起動時・スリープ復帰時に入力 |
| Apple ID | App Store・iCloud・各種サービスで使用 |
| キーチェーン | Mac 標準のパスワード管理機能。Safari のパスワードを自動保存 |
| Touch ID | 指紋認証でパスワード入力を省略 |

> **Touch ID の設定**: `システム設定` → `Touch ID とパスコード` で指紋を登録。
> ログイン・App Store 購入・パスワード自動入力に使える。

## 12. バックアップ

### Time Machine

Mac 標準のバックアップ機能。外付けドライブに自動でバックアップを取り続ける。

```
システム設定 → 一般 → Time Machine
```

### 仕組み

| 項目 | 内容 |
|------|------|
| 何をバックアップ | Mac 全体（アプリ・設定・ファイルすべて） |
| 頻度 | 1 時間ごとに自動 |
| 保存期間 | 24 時間分の毎時バックアップ・1ヶ月分の毎日バックアップ・容量が許す限りの毎週バックアップ |
| 必要なもの | **外付けドライブ**（USB / Thunderbolt） |

### 設定手順

```
1. 外付けドライブを Mac に接続
2. 「Time Machine バックアップとして使用しますか？」と聞かれるので「使用」
   （聞かれない場合: システム設定 → 一般 → Time Machine → バックアップディスクを追加）
3. 初回バックアップが自動で開始される（時間がかかる）
4. 以降は接続するたびに自動バックアップ
```

### ファイルの復元

```
1. メニューバーの Time Machine アイコン → 「Time Machine に入る」
2. タイムラインを遡って、復元したいファイルを見つける
3. 「復元」をクリック
```

> Windows の「ファイル履歴」「システムの復元」に相当するが、Time Machine はより包括的。
> **外付けドライブを 1 つ持っておくだけで Mac 全体が守られる。**

## 13. トラブルシューティング

### アプリがフリーズした

| 手順 | 操作 | 説明 |
|------|------|------|
| 1 | `Cmd + Option + Esc` | 強制終了ダイアログを表示 |
| 2 | フリーズしたアプリを選択 →「強制終了」 | アプリだけを終了 |
| 3 | それでもダメなら  → 強制終了 | 同じダイアログ |

### Mac 全体がフリーズした

| 手順 | 操作 | 説明 |
|------|------|------|
| 1 | 数分待つ | 重い処理中の可能性がある |
| 2 | 電源ボタンを長押し（10秒） | **強制シャットダウン**（最終手段） |
| 3 | 数秒待ってから電源ボタンで起動 | 再起動 |

### ディスク容量が足りない

| 操作 | 方法 |
|------|------|
| 容量の確認 |  →「この Mac について」→「ストレージ」 |
| ストレージ管理 |  →「この Mac について」→「ストレージ」→「管理」 |
| 大きなファイルを探す | Finder → `Cmd + F` → 「ファイルサイズ」で絞り込み |
| ゴミ箱を空にする | Dock のゴミ箱を右クリック →「ゴミ箱を空にする」 |
| ダウンロードフォルダの整理 | 不要なインストーラー（`.dmg`）を削除 |

> **見落としがちな容量消費**: ゴミ箱に入れたファイルは容量を消費し続ける。
> 定期的に「ゴミ箱を空にする」を実行すること。

### セーフモード（セーフブート）

問題の切り分けに使う起動モード。最小限のドライバ・拡張機能だけで起動する。

#### Apple Silicon Mac（M1 / M2 / M3 / M4）の場合

```
1. Mac をシャットダウン
2. 電源ボタンを長押し（「起動オプションを読み込み中」が表示されるまで）
3. 起動ディスクを選択
4. Shift キーを押しながら「セーフモードで続ける」をクリック
```

#### Intel Mac の場合

```
1. Mac をシャットダウン
2. 電源ボタンを押してすぐ Shift キーを押し続ける
3. ログイン画面が表示されたら Shift を離す
```

> セーフモードでは画面右上に「セーフブート」と表示される。
> 問題がセーフモードで発生しない場合、サードパーティのソフトが原因の可能性が高い。

### その他の対処法

| 症状 | 対処法 |
|------|--------|
| Wi-Fi がつながらない | Wi-Fi をオフ → オン / ルーターを再起動 |
| Bluetooth が不安定 | `Option` + メニューバーの Bluetooth アイコン →「デバッグ」→「すべてのデバイスを削除」→ 再ペアリング |
| 動作が重い | アクティビティモニタ（Spotlight で `activity`）で CPU / メモリ使用量を確認 |
| 音が出ない | システム設定 → サウンド → 出力デバイスを確認 |
| 外部ディスプレイが映らない | ケーブルの抜き差し / システム設定 → ディスプレイ →「ディスプレイを検出」 |
