### **プロジェクト企画書（最終版）：『Project Compass AI』**
**フィロソフィー：あたり前の積み重ねで、特別なものを。**
**コンセプト：世界最高のヘッジファンドを、標準ツールとAIでシミュレートする。**

#### **1. エグゼクティブ・サマリー（プロジェクトビジョン）**

本プロジェクトは、Googleの標準ツール群（Sheets, GAS, Colab, Docs, Gemini）と、金融業界で確立された専門的ナレッジフレームワークを体系的に組み合わせる。その上で、**「世界最高峰のヘッジファンドの組織構造と日々の活動」**というアナロジーを導入する。これにより、個人の開発者が、あたかも専門家チームを率いるポートフォリオ・マネージャー（PM）のように、高度なデータ分析とAIによる洞察に基づいた意思決定を行える、パーソナル・インテリジェンス・エンジンを構築することを目的とする。

#### **2. Guiding Principles（行動指針）**

1.  **徹底的なアクセシビリティと低コスト:** 全てのツールは標準的で無料、または極めて少額なものに限定する。複雑な環境構築や高価なライセンスは一切不要とする。
2.  **相乗効果の追求:** システムの価値は、単一の高性能ツールに依存しない。平易なツール群を「構造的に組み合わせる」ことで、プロ級のアウトプットという「特別な価値」を生み出す。
3.  **透明性と再現性:** 全てのデータとロジックはGoogle SheetsやColab上で「見える化」され、ブラックボックスを防ぐ。AIへの「問い」もテンプレート化し、思考の再現性を担保する。

#### **3. ヘッジファンド・アナロジー：組織と役割**

本システムは、以下の専門家チームとして機能する。**あなた自身が、このチームを率いるポートフォリオ・マネージャー（PM）である。**

| ヘッジファンドの役割 | 担当ツール/プロセス | システム内での責務 |
| :--- | :--- | :--- |
| **ポートフォリオ・マネージャー (PM)** | **あなた自身** | 全ての報告を受け、最終的な問いを発し、意思決定を行う。 |
| **クオンツ・アナリスト (Quants)** | **Google Colaboratory** | R&Dに特化し、客観的な事実と統計的優位性を探る「思考の実験室」。 |
| **データ・エンジニア (Data Eng)** | **Google Apps Script (GAS)** | 毎晩、市場からデータを自動で収集・整備するシステムの「神経系」。 |
| **ジュニア・アナリスト** | **Google Sheets (関数)** | データを基に相関等の基礎集計を行う「コックピット」の計器盤。 |
| **マクロ・ストラテジスト** | **Gemini (Web検索機能付き)** | 世界中のニュースやレポートを読み解き、市場の「物語」を提供する「目と耳」。 |
| **チーフ・インベストメント・オフィサー (CIO)** | **Gemini (API経由の統合分析)** | 全情報を統合し、PMへ最終推奨メモを提出する「最高の頭脳」。 |
| **アーキビスト（記録係）** | **Google Docs** | 全ての分析と意思決定を記録し、組織の「記憶」を構築する。 |

#### **4. システムアーキテクチャとコスト評価**
| コンポーネント | 利用ツール | 役割 | コスト評価 |
| :--- | :--- | :--- | :--- |

| **DB・UI** | Google Sheets | データの格納・可視化 | **無料** |
| **外部インテリジェンス** | Gemini (Web検索機能付き) | 市場コンセンサスの収集 | **無料 / 少額** |
| **統合分析・洞察** | Gemini API | 最終レポートの生成 | **少額** |
| **最終報告・ナレッジ蓄積**| Google Docs | 知見のアーカイブ | **無料** |


#### **5. 黄金のワークフロー：ヘッジファンドの1日**

1.  **【夜間/定時】データ収集 (データエンジニアの活動)**
    *   GASが全データを自動更新。

2.  **【早朝】定量的ブリーフィング (ジュニア・アナリストの報告)**
    *   PM（あなた）がSheetsのコックピットを確認。相関やリターンなどの「事実」が構造化テキストとして`AI_Input`シートに自動で準備される。

3.  **【日中】インテリジェントな問い (PMからCIOへの分析指令)**
    *   PM（あなた）は、準備された「事実」と、事前に設計した「黄金のプロンプト」を組み合わせ、CIO（Gemini API）に分析を指令する。
        *   **指令の核心:** 「内部データからの示唆はこうだ。外部のコンセンサスを調査し、両者を比較した上で、今日の最終見解を述べよ。」

4.  **【即時】統合レポート (CIOからのインベストメント・メモ)**
    *   CIO（Gemini）が、内部シグナルと外部コンセンサスの比較分析を含む、質の高いレポートをダッシュボード上に提出する。

#### **6. WBS (Work Breakdown Structure) - ヘッジファンド組織の構築計画**

##### **フェーズ1：コックピットの設営**
*   `1.1` PM用ダッシュボードの基本設計 (Sheets)
*   `1.2` ジュニア・アナリストの採用 (関数による基礎集計)

##### **フェーズ2：クオンツ部門の設立 (R&D)**
*   `2.1` R&Dラボの開設 (Colab)
*   `2.2` 分析フレームワークの歴史データによるバックテスト

##### **フェーズ3：オペレーション部門の構築とCIOの育成**
*   `3.1` データエンジニアリングチームの配備 (GASによる自動化)
*   `3.2` マクロ・ストラテジストの調査能力の検証 (Web検索付きAIの活用)
*   `3.3` CIOへの最高の質問術（プロンプト）の確立 (Google AI Studio)

##### **フェーズ4：完全統合とアーカイブ化**
*   `4.1` CIOのAPI統合 (GASとGemini APIの連携)
*   `4.2` 投資メモの自動提出システムの導入 (ダッシュボードへの書き込み)
*   `4.3` アーキビストの配備 (Google Docsへの自動記録)


#### **7. プロジェクト評価：客観的視点**

*   **強み (Strengths):** 卓越したコンセプト、完璧な技術選定、洗練されたワークフロー、現実的なロードマップ。アイデアと構造で勝負する理想的な設計。
*   **弱み (Weaknesses):** 外部データソースへの依存、AIの非決定性（ブラックボックス性）、プロンプトエンジニアリングの高い難易度。
*   **機会 (Opportunities):** AIの急速な進化を直接システムの強化に繋げられる将来性。自身の投資スタイルに合わせた無限の拡張性。
*   **リスク (Risks):** AIレポートの過信。対策として、常に「事実」と「解釈」を並べて表示し、PM（あなた）が最終判断を下すプロセスを徹底する。

#### **8. 結論**

本企画書は、単なるツールの開発計画ではない。それは、**世界最高峰のヘッジファンドという「思考のOS」を、誰もが使える標準ツールの上に実装する**という壮大な試みである。各ツールを専門家として擬人化し、日々の活動をシミュレートすることで、我々は単なる情報ではなく、構造化された「インテリジェンス」を生み出す。この「あたり前の積み重ね」の先には、市場をより深く理解し、より賢明な意思決定を下すための、あなただけの「特別な」羅針盤が完成するだろう。
