### **主要投資戦略 体系的解説表**

| 戦略カテゴリー (Category) | 戦略名 (Strategy Name) | 哲学 (Philosophy / Core Concept) | 手法 (Methodology) | 最適な市場環境 (Optimal Market Environment) |
| :--- | :--- | :--- | :--- | :--- |
| **I. マクロ主導型**<br>(Macro-Driven) | **グローバル・マクロ** | 経済の大きな物語は予測可能であり、それに基づいて収益を上げる。 | 金融政策や地政学リスクを分析し、あらゆる資産クラスで裁量的にロング/ショートを行う。 | 金融危機、政策の大きな転換期など、明確なマクロテーマが存在する局面。 |
| | **トレンドフォロー (CTA)** | 一度発生したトレンドは、それが何であれ、しばらく継続する。 | 定量モデルに基づき、上昇トレンドの資産を買い、下降トレンドの資産を売る。人間の判断を介さない。 | 長期間にわたって一方向に進むトレンド相場。（例: 2008年、2022年） |
| **II. リスク配分型**<br>(Risk-Allocation) | **伝統的バランス (60/40)** | 株式は成長を、債券は安定とヘッジを提供する。両者は相互に補完する。 | 資産の60%を株式に、40%を債券に配分し、定期的にリバランスする。 | 安定成長と低インフレが両立する「適温相場（ゴルディロックス）」。 |
| | **リスクパリティ** | ポートフォリオのリスクは、ドル建ての配分額ではなく、各資産の「リスク貢献度」で均等にすべきだ。 | 低ボラティリティの債券などにレバレッジをかけ、各資産のリスク貢献度が均等になるよう調整する。 | 金利が低下または安定し、経済が緩やかに成長する局面。 |
| | **バーベル戦略** | 未来は予測不能。「平均的な未来」を想定した中リスク資産は、予測不能なショックに最も脆弱だ。 | 資産を、①極めて安全な資産と、②極めて積極的な資産の両極端に集中させ、「真ん中」を持たない。 | **不確実性が高く、二極化した市場。**（まさに現在の市場環境） |
| **III. ファクター/スタイル**<br>(Factor/Style) | **バリュー (Value)** | 市場は時として、企業の真の価値より安く値付けする。 | 低PBR、低PERなど、本質的価値に比べて割安な銘柄に投資する。 | 景気回復の初期段階。金利上昇局面。 |
| | **グロース (Growth)** | 将来の成長性こそが株価の最大のドライバーである。 | 高い売上成長率や利益成長率を持つ企業に投資する。 | 低金利環境。技術革新が起きている局面。 |
| | **モメンタム (Momentum)** | 勝者は勝ち続け、敗者は負け続ける傾向がある。 | 直近で価格が上昇している銘柄を買い、下落している銘柄を売る。 | 明確なトレンドが存在する市場。 |
| | **低ボラティリティ (Low Volatility)** | 値動きが安定した企業は、長期的にリスク調整後リターンで優れる。 | ボラティリティの低い銘柄群に投資する。 | 市場の不確実性が高い局面。リスクオフのセンチメント。 |
| **IV. ヘッジ/オルタナティブ**<br>(Hedge/Alternative) | **ロング・ショート株式** | 市場全体の方向性（ベータ）ではなく、銘柄選別能力（アルファ）で収益を上げる。 | 割安と判断した銘柄を買い（ロング）、同時に割高と判断した銘柄を空売り（ショート）する。 | 個別企業間で業績の差が大きく開く「分散の大きい」市場。 |
