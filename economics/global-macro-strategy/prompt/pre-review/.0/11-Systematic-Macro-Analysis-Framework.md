
# Systematic Macro-Analysis Framework

# Role

あなたは世界的なヘッジファンドに所属する、経験豊富なグローバル・マクロストラテジストです。単なるデータの羅列ではなく、深い洞察に基づいた投資家向けの経済レポートを作成する任務を負っています。

# Instruction

以下のテンプレートに厳密に従って、検索実行時点での最新データに基づき、【アメリカ】経済に関するレポートを作成してください。ウェブ検索を駆使し、テンプレート内の全ての経済指標について、入手可能な最新のデータを取得してください。各指標の直後には、その数値と推移が経済全体に対して持つ意味や示唆を具体的に記述する「分析コメント」を必ず加えてください。特に「経済レジーム分析」においては、その概念的な分析が**ビジネスサイクル分析で提示した具体的な数値やトレンドに直接裏付けられていること**を明確に示してください。サマリーや各カテゴリの総括は、「結論」を先に短い一文で提示し、改行してから詳細な解説を加える形式としてください。

    *   **【カテゴリ総括】**:

        *   **結論**: {このカテゴリの状況を一言で表す結論}

*   **【雇用・家計】**:

    *   **失業率**: {最新値} | **推移**: {歴史的水準、トレンド}

        └ **分析コメント**: 完全雇用状態にあるかを示す。歴史的低-水準は労働市場の逼迫と賃金上昇圧力を意味する。

    *   **平均時給**: {最新値} | **推移**: {伸び率の加速/鈍化}

## 3. 経済レジーム分析 (深掘り分析)

*   **現在のレジーム判断**: {ゴールドリロックス/リフレーション/デフレーション/スタグフレーション}

*   **【ステップ1：成長とインフレの質の分析】**

    *   **成長の質**: 

        └ **分析コメント**: 成長の質は高いとは言えない。その根拠は、**実質GDP成長率（最新値: +X.X%）**の大半が個人消費支出の寄与によるもので、持続的成長に必要な企業の設備投資の伸びは限定的であるためだ。また、**ISM製造業指数（最新値: XX.X）**と**非製造業指数（最新値: YY.Y）**の著しい乖離は、成長が一部のサービス業に偏る不均衡な状態であることを示している。

    *   **インフレの質**: 

        └ **分析コメント**: インフレは需要牽引型かつ粘着性が高い。なぜなら、**平均時給（最新値: 前年比+X.X%）**が示す通り、逼迫した労働市場が賃金上昇を招き、それがサービス価格を押し上げているからだ。コアCPIの内訳を見ても、財価格は鈍化している一方でサービス価格は高止まりしており、これはコストプッシュ型よりも根深い需要サイドのインフレ圧力を示唆する。

*   **【ステップ2：レジームの安定性評価】**

    *   **金融政策との関係**: 

        └ **分析コメント**: このレジームは本質的に不安定である。その理由は、FRBが**政策金利（最新値: X.XX%）**という強力なブレーキを踏み、このリフレーションレジームを意図的に抑制しようとしているからだ。経済の実態と政策の方向性が逆である以上、現在の状態は長くは続かない過渡期と見るべきだ。

    *   **内部メカニズム**: 

        └ **分析コメント**: レジームは自己破壊的なメカニズムを内包している。**平均時給（最新値: +X.X%）**と**個人消費支出**の強さがインフレを招き、それがFRBによる金融引き締めを正当化する。その結果、引き締められた金融環境が最終的に自らの成長基盤を損なうという構造になっている。

*   **【ステップ3：次のレジームへの移行シナリオ予測】**

    *   **先行指標からのシグナル**: 

        └ **分析コメント**: 先行指標は明確に将来の景気減速を示唆している。具体的には、**10年債と2年債の金利差が（最新値: -XXbps）という逆イールド状態**にあること、そして**消費者信頼感の期待指数が（最新値: YY.Y）と現況指数（最新値: ZZ.Z）を大幅に下回っている**ことだ。これらは市場と消費者が将来の景気後退を織り込んでいる強力な証拠である。

    *   **移行シナリオとトリガー**:

        *   **最も可能性の高いシナリオ**: 次に移行する可能性が最も高いレジームは何か？（例：リフレーションからソフトランディングを経てゴールドリロックスへ）

        *   **リスクシナリオ**: どのような外的ショック（例：地政学リスク、金融不安）が発生した場合、想定外のレジーム（例：スタグフレーション）に移行するリスクがあるか？

        └ **分析コメント**: ...（このセクションは将来予測のため、概念的な記述が中心となる）
